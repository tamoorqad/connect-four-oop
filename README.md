# Connect Four - OOP Term Project

Student Name: Tamoor Qadeer
Team: Individual

## Project Description
This project is a console-based implementation of the Connect Four game using Object-Oriented Programming principles.

The game will support:
- Two human players
- 7x6 board
- Win detection (horizontal, vertical, diagonal)
- Input validation
- Game restart functionality

## Planned OOP Structure
- Controller: Manages game flow and turn handling
- GameBoard: Stores and manages the board state
- Player (Abstract Class): Base class for players
- HumanPlayer: Inherits from Player

This repository will be updated progressively according to course milestones.
